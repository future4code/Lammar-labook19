#

### Endpoints: 

- POST Signup
- POST Login
- POST Create Post
- GET Get post by id
- POST Create friendship
- DEL Delete friendship

#

### Ferramentas e Tecnologias Utilizadas:

- Node.js
- TypeScript
- Knex
- Express
- Mysql
